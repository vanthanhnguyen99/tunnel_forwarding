This MVP binds a local port, accepts client connections, and relays each session through an SSH server to the configured remote destination.

  • Local listeners per endpoint with SSH as the transport layer
  • Per-session SSH bridge with byte counters
  • SSE realtime updates for metrics and session events

Default admin credentials come from ADMIN_DEFAULT_USER and ADMIN_DEFAULT_PASS.
